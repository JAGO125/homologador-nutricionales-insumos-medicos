# Homologador Inteligente de Insumos y MAOS

Motor de homologación automática de productos nutricionales e insumos médicos.

## Funcionalidades

- Limpieza automática de descripciones
- Normalización de texto
- Homologación mediante fuzzy matching
- Validación estructural
- Generación de descripciones estandarizadas
- Descarga automática de base SQL

## Tecnologías

Python  
Tkinter  
Pandas  
RapidFuzz  
SQL Server  

## Uso

```bash
python homologador_nutricionales_insumos_medicos.py
